# frontend

A new Flutter project.

HOW to run the app
1. open xampp
2. start apache and mysql
3. add terminal (cmd)
4. run: venv\Scripts\activate.bat
5. cd backend
6. run: uvicorn app.main:app --reload
7. go to other terminal for frontend
8. cd frontend
9. flutter run -d edge
